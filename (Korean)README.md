## 코드에서의 변경사항 

As you can see, Even though they made their own website, doesn't put contents in there. 
So users can be confused about it. That's why I combined Those two website's **strength**. 
I changed this website into resposive webpage. 
Because They do not have application so They really need a responsive webpage for mobile users. 
I made breakpoints : 

(min-width:1024px) and (max-width:1365px)  /* Tablets in landscape mode, older desktop monitors */ <br>
(min-width:768px) and (max-width:1023px)  /* Tablets in portrait mode, large display smartphones landscape mode*/<br>
(max-width:420px) /* Smartphones in portrait mode */
